# IT0079 Android Reviewer

A single-page quiz reviewer for IT0079 Formative 3 and Formative 4.

## Features
- Immediate answer feedback
- Incorrect choices turn red
- Correct choices turn green
- Users can keep trying until they choose the correct answer
- Previous and Next buttons
- Number grid for jumping to any question
- FA3, FA4, and All Questions modes
- Local progress saving
- Shuffle and reset controls

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` to the root of the repository.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.
6. Wait briefly for the GitHub Pages link to appear.

## Notes
The reviewer is fully contained in `index.html`. No installation or external libraries are required.
